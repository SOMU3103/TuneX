Download Instructions

Follow these simple steps to get TuneX up and running on your device

1

Install FFmpeg

FFmpeg is required for TuneX to work properly. Follow these steps:

  1. Click this link: FFmpeg Releases
  2. Download the Windows version (ffmpeg-master-latest-win64-gpl-shared.zip)
  3. Extract the ZIP file to a location like C:\FFmpeg
  4. Copy the path to the "bin" folder (e.g., C:\FFmpeg\bin)
  5. Add this path to your system's Environment Variables
üí° FFmpeg is essential for audio processing in TuneX
